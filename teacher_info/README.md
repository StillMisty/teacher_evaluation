## 详细信息 info

teacherInfo里的academicTitle是职称

* 1：教授
* 2: 副教授
* 3: 讲师
* 4：研究员
* 5：副研究员
* 6：助理研究员
* 7：研究实习员
* 30：助教

teacherInfo里的updateTime是数据更新时间

一定要注意：它返回的数据有的有，有的没有，你一定要判断一下，不然会报错
